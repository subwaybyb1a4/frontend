# 🚇 편안한 지하철 내비게이션 (easy-subway)

지하철 내비게이션 프로젝트의 프론트엔드 저장소입니다. 소요 시간 단축보다 '사람에 끼지 않고 편하게' 가는 경로를 추천하는 서비스를 목표로 합니다.

## 🎨 Design Resources

- **Information Architecture:** [Figma Design Link](https://www.figma.com/design/2UcELfKPSumFs4Wmqpl17X/Information-Architecture-Design)

## 🚀 시작하기

이 프로젝트는 [Expo](https://expo.dev)로 생성되었습니다.

1. **의존성 설치**
   ```bash
   npm install
   ```
